# Betreffzeilen + Preheader (alle 6 Sprachen)

Pro Sprache: drei Subject-Line-Varianten zum A/B-Testen, plus
einen Preheader-Text der unter dem Betreff in der Inbox erscheint.

CleverReach erlaubt Subject-Line-A/B-Tests direkt im Mailing-Editor.

---

## 🇩🇪 Deutsch

**Subject A** (direkt + Nutzen):
```
Bootseigner suchen Service in {$city} — werden Sie gefunden?
```
**Subject B** (kurz + Skipily-Brand):
```
Skipily: Mehr Sichtbarkeit für Ihre Werft
```
**Subject C** (persönlich + Frage):
```
Hallo {$company}, dürfen wir Sie Bootseignern vorstellen?
```

**Preheader:**
```
Bootseigner suchen Werkstätten, Segelmacher und Yachtservice direkt in der Nähe — bei Skipily zeigen Sie sich allen, die ein Boot in Ihrer Region haben.
```
