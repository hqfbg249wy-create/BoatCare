HTML"**-Blöcke eingefügt.

## Mehrsprachigkeit in WordPress einrichten
Empfohlen: **Polylang** (kostenlos) oder **TranslatePress**.
1. Plugin installieren, Sprachen **Deutsch (Standard)** + **English** anlegen.
2. Für jede Seite eine englische Übersetzung erstellen → den Inhalt der
   passenden Datei oben in einen „Benutzerdefiniertes HTML"-Block einfügen.
3. Sprachumschalter (Flaggen/Menü) ins Header-Menü einbinden (Polylang-Widget
   bzw. Block).
4. Permalinks prüfen: z. B. `skipily.app/en/`, `skipily.app/en/privacy` etc.
5. Newsletter-/Formular-Shortcode auf der EN-Home wie auf der DE-Seite einsetzen.

## ⚠️ Rechtstexte
Impressum, Datenschutz und AGB sind **Höflichkeits-Übersetzungen**. Die
**deutsche Fassung bleibt rechtsverbindlich** (Hinweis steht in jeder Datei).
Vor Veröffentlichung bitte **anwaltlich prüfen** lassen — besonders DSGVO und
Impressumspflicht. Platzhalter (`[Vor- und Nachname]` etc.) im Impressum/
Datenschutz müssen wie in der deutschen Fassung gefüllt werden.

## Hinweis App-Stores

die Android-App live ist, „App Store" → „App Store & Google Play" ergänzen.
